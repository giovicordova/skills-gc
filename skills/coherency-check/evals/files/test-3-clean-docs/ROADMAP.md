# Roadmap

## v1.0 — Core Metronome (current)

- [x] Tempo control (20–300 BPM)
- [x] Time signature selection (4, 8, 16 denominators)
- [x] Accent pattern editor
- [x] Tap tempo
- [x] Preset save/recall (up to 20 presets)
- [x] Mobile-responsive layout

## v1.1 — Practice Tools

- [ ] Tempo ramp: gradually increase BPM over a set period
- [ ] Practice timer: count down a session with automatic preset switching
- [ ] Subdivision toggle: add 8th or 16th note subdivisions to the click

## v2.0 — Collaboration

- [ ] Share presets via URL
- [ ] Multi-device sync (WebSocket)
- [ ] Band mode: different click sounds per instrument section
