# Product Vision

## Purpose

Metronome is a practice tool for musicians. It provides a configurable metronome with tempo, time signature, and accent pattern controls.

## Target Users

- Music students practising scales and exercises
- Gigging musicians needing a reliable click track
- Drum teachers demonstrating time signatures

## Core Features

1. **Tempo control** — 20 to 300 BPM, adjustable in 1 BPM increments
2. **Time signatures** — common (4/4, 3/4, 6/8) and custom (any X/Y where Y is 4, 8, or 16)
3. **Accent patterns** — first beat always accented, additional accents configurable
4. **Tap tempo** — tap a button to set BPM from feel
5. **Presets** — save and recall tempo + time signature combinations
